## About
FlyXpert is an academic project built for the OOP (Object-Oriented Programming) course.  
This desktop application implements a flight booking system that allows users to:  
- Sign up / Sign in  
- Search and filter flights  
- Book flights for themselves and/or other passengers  
- Choose seats for each passenger  
- Pay using preferred payment method  
- Manage their bookings  

In addition, administrators have a special interface which lets them:  
- Add new flights  
- Edit existing flight details  
- Delete flights  

This project was designed and implemented by Tamana & Anwesha to demonstrate mastery of OOP principles, GUI design, and desktop app development.

---

## Features
- User registration and login  
- Flight search & filtering by date, destination, etc.  
- Multi-passenger booking support  
- Seat selection interface  
- Payment simulation (mock)  
- Booking history and management  
- Admin panel for flight management  

---



## Technology Stack
- Java (core language)  
- JavaFX (desktop GUI)  
- OOP concepts: classes, inheritance, polymorphism, encapsulation  
- Data stored in text files: FlightInformation.txt, BookingRecords.txt, Users.txt  
- Maven project structure (`pom.xml`)
